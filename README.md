# Password Strength Checker

Python program to check password strength (Weak, Medium, Strong) based on rules: length, uppercase, lowercase, number, and special character. Provides feedback to improve weak passwords.

Usage:

```bash
python password.py
```

Example Output:
```
Enter password: abc123
Password Strength: Medium
Suggestions:
- Add at least 8 characters
- Add an uppercase letter
- Add a special character
```

Tech: Python 3, Regex (re module)
